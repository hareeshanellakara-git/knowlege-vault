# Knowledge Vault

Knowledge Vault is a student-focused web application designed to help users organize study materials, track exam preparation, and manage academic resources efficiently in a structured and distraction-free environment.

## Overview

The application provides a centralized dashboard where students can monitor progress, manage subjects, and maintain important resources. The focus of this project is clean UI design, usability, and logical frontend structure.

## Key Features

- Interactive Dashboard with progress tracking  
- Exam Tracker with subject-wise organization  
- Resource Upload and Bookmark management  
- Focus section for priority-based study  
- Help and FAQ section with accordion functionality  
- Fully responsive design  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript  


## Future Improvements
