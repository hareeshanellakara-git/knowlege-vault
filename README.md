# 📚 Knowledge Vault

Knowledge Vault is a full-stack student productivity web application designed to help users organize study resources, track exam preparation, and stay consistent with their learning — all in one structured platform.

It combines smart tracking, real-time analytics, and resource management with a clean and modern user interface.

---

## 🎯 Overview

Knowledge Vault provides a centralized academic workspace where students can:

- Manage subjects and exam schedules
- Track preparation progress
- Maintain study consistency with streak tracking
- Organize learning resources efficiently

The project focuses on **real-world usability, responsive design, and dynamic data handling using a backend system**.

## ⚙️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

### Backend
- Node.js  
- Express.js  
- JSON-based data storage  
- REST API (Fetch integration)

---

## 🔄 How It Works

- Frontend communicates with backend using `fetch()`
- Backend provides data through API endpoints (`/data`)
- Subjects, progress, streak, and activities are stored in JSON
- Dashboard dynamically updates in real-time based on backend data
