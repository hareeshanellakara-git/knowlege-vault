Help & User Guide

How to Use Knowledge Vault

  1. Navigate to the Exam Tracker to add subjects and topics.
  2. Mark topics as completed using checkboxes.
  3. View progress analytics on the Dashboard.
  4. Upload study materials in Upload Notes.
  5. Save useful links under Bookmarks.

Feature Guidance

  • Upload PDFs and notes for centralized access.
  • Track syllabus completion subject-wise.
  • Monitor preparation using dashboard insights.
  • Bookmark learning playlists and articles.
